# SteamCMD GUI 简体中文汉化版

*为 **[SteamCMD]** 制作的图形界面.*

<br>
<br>

[![Button Tutorials]][Tutorials]   
[![Button Authors]][Authors]   
[![Button Changelog]][Changelog]

<br>
<br>

SteamCMD GUI提供了一个简单直观的图形界面，可用作批处理文件和使用命令行的替代方案来使用 **[SteamCMD]**。

它也可以用于运行服务器以及在只需几个点击即可打开和编辑文件，如`MOTD`或`maplist`。
